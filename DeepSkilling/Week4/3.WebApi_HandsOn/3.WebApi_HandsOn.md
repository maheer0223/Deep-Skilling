# 3.WebApi_HandsOn - Custom Models, Action & Exception Filters in ASP.NET Core Web API

## Objectives & Conceptual Overview

### 1. Custom Model Classes & Complex Entity Graphs
- **Entity Model Structure:**
  In enterprise APIs, entities often consist of nested model graphs. The `Employee` class contains complex properties such as a nested `Department` object and a `List<Skill>` collection along with scalar types (`int`, `string`, `bool`, `DateTime`).
- **`[AllowAnonymous]` Attribute:**
  Action methods decorated with `[AllowAnonymous]` bypass global or controller-level authorization action filters (such as `CustomAuthFilter`). This allows public access to specific endpoints like `/api/Employee/standard` while securing other endpoints.
- **`[ProducesResponseType]` Attribute:**
  Explicitly documents HTTP status codes (e.g., `200 OK`, `201 Created`, `400 Bad Request`, `404 Not Found`, `500 Internal Server Error`) and response body types in the OpenAPI / Swagger documentation schema.

---

### 2. Usage & Role of the `[FromBody]` Attribute
- **Body Parameter Binding:**
  By default, primitive types are bound from query string parameters. Complex objects passed to API actions should be read directly from the HTTP request body (`application/json`).
- **`[FromBody]` Mechanism:**
  Decorating an action parameter with `[FromBody]` instructs the ASP.NET Core model binder to read the incoming JSON payload from the request stream and deserialize it into the target object (`Employee`).
- **Query String vs Request Body:**
  Query string parameters expose sensitive data in the URL and have length limitations. `[FromBody]` securely processes large, structured, or nested JSON payloads in HTTP `POST` and `PUT` requests.

---

### 3. Custom Action Filter for Authorization (`CustomAuthFilter`)
- **Action Filters (`ActionFilterAttribute`):**
  Action filters run before and after an action method executes. Inheriting `ActionFilterAttribute` and overriding `OnActionExecuting` allows developers to inspect incoming HTTP requests.
- **Header Inspection:**
  `CustomAuthFilter` checks the `HttpContext.Request.Headers` for an `Authorization` header key.
  - **Missing Authorization Header:** Sets `context.Result = new BadRequestObjectResult("Invalid request - No Auth token")` (HTTP 400).
  - **Missing 'Bearer' Keyword:** If present but lacking the `Bearer` scheme keyword, sets `context.Result = new BadRequestObjectResult("Invalid request - Token present but Bearer unavailable")` (HTTP 400).

---

### 4. Custom Exception Filter (`CustomExceptionFilter`) & Compatibility
- **Exception Filters (`IExceptionFilter`):**
  Classes implementing `IExceptionFilter` override `OnException(ExceptionContext context)` to intercept unhandled exceptions thrown across action methods.
- **File System Exception Logging:**
  `CustomExceptionFilter` extracts exception metadata (`Type`, `Message`, `StackTrace`, timestamp) and appends it to a physical file on the system (`logs/exceptions.log`).
- **Standardized Error Response:**
  Sets `context.Result` to an `ObjectResult` returning HTTP 500 Internal Server Error and marks `context.ExceptionHandled = true`.
- **`Microsoft.AspNetCore.Mvc.WebApiCompatShim` Note:**
  In legacy ASP.NET Web API 2, `HttpResponseException` and Web API filters relied on `WebApiCompatShim`. In modern ASP.NET Core 8.0+, native MVC filter interfaces (`ActionFilterAttribute`, `IExceptionFilter`) provide clean, native request interception and exception handling without legacy dependencies.

### 4. `Filters/CustomAuthFilter.cs`
```csharp
using Microsoft.AspNetCore.Authorization;
using Microsoft.AspNetCore.Mvc;
using Microsoft.AspNetCore.Mvc.Filters;
using System;
using System.Linq;

namespace WebApiHandsOn3.Filters
{
    public class CustomAuthFilter : ActionFilterAttribute
    {
        public override void OnActionExecuting(ActionExecutingContext context)
        {
            // Allow anonymous access if [AllowAnonymous] attribute is present
            bool hasAllowAnonymous = context.ActionDescriptor.EndpointMetadata
                .OfType<AllowAnonymousAttribute>()
                .Any();

            if (hasAllowAnonymous)
            {
                base.OnActionExecuting(context);
                return;
            }

            var request = context.HttpContext.Request;

            // Check if 'Authorization' header exists
            if (!request.Headers.ContainsKey("Authorization"))
            {
                context.Result = new BadRequestObjectResult("Invalid request - No Auth token");
                return;
            }

            string authHeader = request.Headers["Authorization"].ToString();

            // Check if header value contains 'Bearer'
            if (!authHeader.Contains("Bearer", StringComparison.OrdinalIgnoreCase))
            {
                context.Result = new BadRequestObjectResult("Invalid request - Token present but Bearer unavailable");
                return;
            }

            base.OnActionExecuting(context);
        }
    }
}
```

### 5. `Filters/CustomExceptionFilter.cs`
```csharp
using Microsoft.AspNetCore.Http;
using Microsoft.AspNetCore.Mvc;
using Microsoft.AspNetCore.Mvc.Filters;
using System;
using System.IO;

namespace WebApiHandsOn3.Filters
{
    public class CustomExceptionFilter : IExceptionFilter
    {
        private readonly string _logFilePath;

        public CustomExceptionFilter()
        {
            string logDir = Path.Combine(Directory.GetCurrentDirectory(), "logs");
            if (!Directory.Exists(logDir))
            {
                Directory.CreateDirectory(logDir);
            }
            _logFilePath = Path.Combine(logDir, "exceptions.log");
        }

        public void OnException(ExceptionContext context)
        {
            var exception = context.Exception;
            string timestamp = DateTime.Now.ToString("yyyy-MM-dd HH:mm:ss");

            string logEntry = $"[{timestamp}] EXCEPTION CAUGHT BY CustomExceptionFilter:\n" +
                              $"Type: {exception.GetType().FullName}\n" +
                              $"Message: {exception.Message}\n" +
                              $"StackTrace: {exception.StackTrace}\n" +
                              new string('-', 60) + "\n";

            // Write exception details to file
            File.AppendAllText(_logFilePath, logEntry);

            // Return 500 Internal Server Error response
            context.Result = new ObjectResult(new
            {
                StatusCode = StatusCodes.Status500InternalServerError,
                Message = "An unexpected error occurred. Logged by CustomExceptionFilter.",
                DetailedError = exception.Message
            })
            {
                StatusCode = StatusCodes.Status500InternalServerError
            };

            context.ExceptionHandled = true;
        }
    }
}
```

### 6. `Controllers/EmployeeController.cs`
```csharp
using Microsoft.AspNetCore.Authorization;
using Microsoft.AspNetCore.Http;
using Microsoft.AspNetCore.Mvc;
using System;
using System.Collections.Generic;
using System.Linq;
using WebApiHandsOn3.Filters;
using WebApiHandsOn3.Models;

namespace WebApiHandsOn3.Controllers
{
    [Route("api/[controller]")]
    [ApiController]
    [CustomAuthFilter]
    [TypeFilter(typeof(CustomExceptionFilter))]
    public class EmployeeController : ControllerBase
    {
        private static List<Employee> _employees = new List<Employee>();

        public EmployeeController()
        {
            if (!_employees.Any())
            {
                _employees = GetStandardEmployeeList();
            }
        }

        private List<Employee> GetStandardEmployeeList()
        {
            return new List<Employee>
            {
                new Employee
                {
                    Id = 1,
                    Name = "John Doe",
                    Salary = 75000,
                    Permanent = true,
                    Department = new Department { Id = 101, Name = "IT Infrastructure" },
                    Skills = new List<Skill>
                    {
                        new Skill { Id = 1, Name = "C#" },
                        new Skill { Id = 2, Name = "ASP.NET Core" },
                        new Skill { Id = 3, Name = "SQL Server" }
                    },
                    DateOfBirth = new DateTime(1995, 5, 15)
                },
                new Employee
                {
                    Id = 2,
                    Name = "Jane Smith",
                    Salary = 85000,
                    Permanent = true,
                    Department = new Department { Id = 102, Name = "Software Development" },
                    Skills = new List<Skill>
                    {
                        new Skill { Id = 4, Name = "Angular" },
                        new Skill { Id = 5, Name = "TypeScript" },
                        new Skill { Id = 6, Name = "REST Web API" }
                    },
                    DateOfBirth = new DateTime(1992, 8, 22)
                }
            };
        }

        [HttpGet("standard")]
        [AllowAnonymous]
        [ProducesResponseType(StatusCodes.Status200OK, Type = typeof(List<Employee>))]
        public ActionResult<List<Employee>> GetStandard()
        {
            return Ok(GetStandardEmployeeList());
        }

        [HttpGet]
        [ProducesResponseType(StatusCodes.Status200OK, Type = typeof(List<Employee>))]
        public ActionResult<List<Employee>> Get()
        {
            return Ok(_employees);
        }

        [HttpGet("{id:int}")]
        [ProducesResponseType(StatusCodes.Status200OK, Type = typeof(Employee))]
        [ProducesResponseType(StatusCodes.Status404NotFound)]
        public ActionResult<Employee> GetById(int id)
        {
            var emp = _employees.FirstOrDefault(e => e.Id == id);
            if (emp == null) return NotFound($"Employee with Id = {id} not found.");
            return Ok(emp);
        }

        [HttpGet("exception")]
        [ProducesResponseType(StatusCodes.Status500InternalServerError)]
        public ActionResult<List<Employee>> GetWithException()
        {
            throw new InvalidOperationException("Simulated exception in GET action method to test CustomExceptionFilter file logging!");
        }

        [HttpPost]
        [ProducesResponseType(StatusCodes.Status201Created, Type = typeof(Employee))]
        [ProducesResponseType(StatusCodes.Status400BadRequest)]
        public ActionResult<Employee> CreateEmployee([FromBody] Employee emp)
        {
            if (emp == null) return BadRequest("Invalid employee data provided.");
            if (emp.Id <= 0) emp.Id = _employees.Max(e => e.Id) + 1;
            _employees.Add(emp);
            return CreatedAtAction(nameof(GetById), new { id = emp.Id }, emp);
        }
    }
}
```

### 7. `Program.cs`
```csharp
using Microsoft.AspNetCore.Builder;
using Microsoft.Extensions.DependencyInjection;
using Microsoft.Extensions.Hosting;
using Microsoft.OpenApi.Models;
using System;

var builder = WebApplication.CreateBuilder(args);

builder.Services.AddControllers();
builder.Services.AddEndpointsApiExplorer();
builder.Services.AddSwaggerGen(c =>
{
    c.SwaggerDoc("v1", new OpenApiInfo
    {
        Title = "3.WebApi_HandsOn API",
        Version = "v1",
        Description = "ASP.NET Core Web API hands-on demonstrating Employee model, [FromBody], [AllowAnonymous], CustomAuthFilter, CustomExceptionFilter, and Swagger response types."
    });

    c.AddSecurityDefinition("Bearer", new OpenApiSecurityScheme
    {
        Description = "JWT Authorization header using Bearer scheme. Example: \"Bearer 12345abcdef\"",
        Name = "Authorization",
        In = ParameterLocation.Header,
        Type = SecuritySchemeType.ApiKey,
        Scheme = "Bearer"
    });

    c.AddSecurityRequirement(new OpenApiSecurityRequirement
    {
        {
            new OpenApiSecurityScheme
            {
                Reference = new OpenApiReference { Type = ReferenceType.SecurityScheme, Id = "Bearer" }
            },
            Array.Empty<string>()
        }
    });
});

var app = builder.Build();

app.UseSwagger();
app.UseSwaggerUI(c =>
{
    c.SwaggerEndpoint("/swagger/v1/swagger.json", "3.WebApi_HandsOn API v1");
    c.RoutePrefix = "swagger";
});

app.UseAuthorization();
app.MapControllers();
app.Run();
```
