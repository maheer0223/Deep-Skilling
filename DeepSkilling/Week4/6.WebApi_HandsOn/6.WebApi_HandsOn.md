# 6.WebApi_HandsOn - Kafka Integration with C# (.NET 8.0)

## Objectives & Conceptual Overview

### 1. Introduction to Apache Kafka
Apache Kafka is an open-source distributed event streaming platform used by thousands of companies for high-performance data pipelines, streaming analytics, data integration, and mission-critical applications. It operates on a pub-sub model optimized for high throughput, scalability, and durability.

### 2. Apache Kafka Architecture
- **Brokers:** A Kafka cluster consists of one or more servers called brokers. Brokers store the stream data and serve read/write requests from clients.
- **Topics:** Events or messages in Kafka are organized and durably stored in topics. A topic is similar to a folder in a filesystem, and the events are the files in that folder.
- **Partitions:** Topics are divided into partitions spread across different brokers. Partitions allow multiple consumers to read a topic in parallel, guaranteeing horizontal scalability and partition-level message ordering.
- **Producers:** Client applications that publish (write) events to Kafka topics.
- **Consumers:** Client applications that subscribe to (read and process) events from topics.
- **Consumer Groups:** Consumers are grouped together under a `GroupId` to read partition data concurrently. Each partition of a topic is consumed by only one member of the group, enabling load balancing.

### 3. Role of Zookeeper
Zookeeper serves as a distributed coordination service for Kafka clusters. It manages configuration details, detects broker failure, maintains topic/partition metadata, elects controller brokers, and coordinates consumer offsets (in older versions). Modern Kafka configurations (KIP-500) support KRaft mode (Kafka Raft Metadata mode) which eliminates the Zookeeper dependency, but Zookeeper is still widely used in production.

## Installation & Configuration Guide for Windows

### Step 1: Download Apache Kafka
1. Download the latest binary release from [Apache Kafka Official Downloads](https://kafka.apache.org/downloads) (e.g., Scala version binary file `kafka_2.13-3.6.1.tgz`).
2. Extract the downloaded `.tgz` archive to a local path (e.g., `C:\kafka`).

### Step 2: Configure Properties
1. Open `C:\kafka\config\zookeeper.properties` and update directory paths:
   ```properties
   dataDir=C:/kafka/zookeeper-data
   ```
2. Open `C:\kafka\config\server.properties` and update log paths:
   ```properties
   log.dirs=C:/kafka/kafka-logs
   ```

### Step 3: Run Zookeeper Server
Open a command prompt in `C:\kafka` and execute:
```cmd
.\bin\windows\zookeeper-server-start.bat .\config\zookeeper.properties
```

### Step 4: Run Kafka Server Broker
Open a new command prompt in `C:\kafka` and execute:
```cmd
.\bin\windows\kafka-server-start.bat .\config\server.properties
```

### Step 5: Create a Topic
Create a new topic named `console-chat-topic` and `winforms-chat-topic`:
```cmd
.\bin\windows\kafka-topics.bat --create --bootstrap-server localhost:9092 --replication-factor 1 --partitions 1 --topic console-chat-topic

.\bin\windows\kafka-topics.bat --create --bootstrap-server localhost:9092 --replication-factor 1 --partitions 1 --topic winforms-chat-topic
```

### 1. `ChatConsoleApp` - Interactive Kafka Chat Terminal
This Console application starts a background consumer task while running a loop on the main thread to publish messages typed by the user to the local Kafka broker.

- **`ChatConsoleApp.csproj`:**
```xml
<Project Sdk="Microsoft.NET.Sdk">
  <PropertyGroup>
    <OutputType>Exe</OutputType>
    <TargetFramework>net8.0</TargetFramework>
    <ImplicitUsings>enable</ImplicitUsings>
    <Nullable>enable</Nullable>
  </PropertyGroup>

  <ItemGroup>
    <PackageReference Include="Confluent.Kafka" Version="2.4.0" />
  </ItemGroup>
</Project>
```

- **`Program.cs`:**
```csharp
using System;
using System.Threading;
using System.Threading.Tasks;
using Confluent.Kafka;

namespace ChatConsoleApp
{
    class Program
    {
        private const string BootstrapServers = "localhost:9092";
        private const string Topic = "console-chat-topic";

        static async Task Main(string[] args)
        {
            Console.WriteLine("================================================================================");
            Console.WriteLine("             APACHE KAFKA CONSOLE CHAT APPLICATION (C# .NET 8.0)                ");
            Console.WriteLine("================================================================================");

            Console.Write("Enter your Nickname to join the chat: ");
            string nickname = Console.ReadLine() ?? "Anonymous";
            
            Console.WriteLine($"\nWelcome, {nickname}! Joining channel: '{Topic}'");
            Console.WriteLine("Connecting to Kafka broker at localhost:9092...");
            Console.WriteLine("Type your message and press Enter. Type '/exit' to quit.\n");

            var producerConfig = new ProducerConfig { BootstrapServers = BootstrapServers };
            var consumerConfig = new ConsumerConfig
            {
                BootstrapServers = BootstrapServers,
                GroupId = "chat-group-" + Guid.NewGuid().ToString(),
                AutoOffsetReset = AutoOffsetReset.Earliest
            };

            var cts = new CancellationTokenSource();

            // Start Consumer background thread
            Task consumerTask = Task.Run(() => StartConsumer(consumerConfig, Topic, nickname, cts.Token));

            // Run Producer loop in main thread
            await StartProducer(producerConfig, Topic, nickname, cts);

            try { await consumerTask; } catch (Exception) {}
            Console.WriteLine("\nKafka Chat Closed!");
        }

        private static async Task StartProducer(ProducerConfig config, string topic, string nickname, CancellationTokenSource cts)
        {
            try
            {
                using var producer = new ProducerBuilder<Null, string>(config).Build();
                while (true)
                {
                    string input = Console.ReadLine();
                    if (string.Equals(input, "/exit", StringComparison.OrdinalIgnoreCase))
                    {
                        cts.Cancel();
                        break;
                    }

                    if (string.IsNullOrWhiteSpace(input)) continue;

                    string messagePayload = $"[{DateTime.Now:HH:mm:ss}] {nickname}: {input}";

                    try
                    {
                        await producer.ProduceAsync(topic, new Message<Null, string> { Value = messagePayload });
                    }
                    catch (ProduceException<Null, string> ex)
                    {
                        Console.ForegroundColor = ConsoleColor.Red;
                        Console.WriteLine($"\n[Error] Delivery failed: {ex.Error.Reason}");
                        Console.ResetColor();
                    }
                }
            }
            catch (Exception ex)
            {
                Console.ForegroundColor = ConsoleColor.Red;
                Console.WriteLine($"\n[Critical Error] Kafka Producer down: {ex.Message}");
                Console.ResetColor();
            }
        }

        private static void StartConsumer(ConsumerConfig config, string topic, string currentNickname, CancellationToken cancellationToken)
        {
            try
            {
                using var consumer = new ConsumerBuilder<Ignore, string>(config).Build();
                consumer.Subscribe(topic);

                while (!cancellationToken.IsCancellationRequested)
                {
                    try
                    {
                        var consumeResult = consumer.Consume(cancellationToken);
                        if (consumeResult != null)
                        {
                            string message = consumeResult.Message.Value;

                            if (!message.Contains($" {currentNickname}: "))
                            {
                                Console.ForegroundColor = ConsoleColor.Cyan;
                                Console.WriteLine(message);
                                Console.ResetColor();
                            }
                            else
                            {
                                Console.ForegroundColor = ConsoleColor.Gray;
                                Console.WriteLine($"{message} (delivered)");
                                Console.ResetColor();
                            }
                        }
                    }
                    catch (Exception) { break; }
                }
                consumer.Close();
            }
            catch (Exception) {}
        }
    }
}
```

### 2. `ChatWinFormsApp` - C# Windows Forms Multi-Client App

```xml
  <ItemGroup>

  </ItemGroup>
</Project>
```

- **`Form1.cs`:**
```csharp
using System;
using System.Drawing;
using System.Threading;
using System.Threading.Tasks;
using System.Windows.Forms;
using Confluent.Kafka;

namespace ChatWinFormsApp
{
    public partial class Form1 : Form
    {
        private IProducer<Null, string>? _producer;
        private CancellationTokenSource? _cts;
        private const string Topic = "winforms-chat-topic";
        private bool _isConnected = false;

        private RichTextBox rtbMessages;
        private TextBox txtMessage;
        private TextBox txtNickname;
        private TextBox txtBroker;
        private Button btnConnect;
        private Button btnSend;

        public Form1()
        {
            InitializeComponent();
            SetupCustomLayout();
        }

        private void SetupCustomLayout()
        {
            this.Text = "Kafka Windows Forms Chat Client";
            this.Size = new Size(680, 480);
            this.BackColor = Color.FromArgb(30, 30, 30);
            this.ForeColor = Color.White;
            this.Font = new Font("Segoe UI", 9.5F);

            // Connect Panel
            Panel pnlConnect = new Panel { Dock = DockStyle.Top, Height = 55, BackColor = Color.FromArgb(45, 45, 45) };
            
            Label lblName = new Label { Text = "Nickname:", Location = new Point(10, 18), AutoSize = true };
            txtNickname = new TextBox { Text = "User_" + Guid.NewGuid().ToString().Substring(0,4), Location = new Point(80, 15), Width = 110, BackColor = Color.FromArgb(60,60,60), ForeColor = Color.White };
            
            Label lblBroker = new Label { Text = "Broker:", Location = new Point(210, 18), AutoSize = true };
            txtBroker = new TextBox { Text = "localhost:9092", Location = new Point(265, 15), Width = 130, BackColor = Color.FromArgb(60,60,60), ForeColor = Color.White };
            
            btnConnect = new Button { Text = "Connect", Location = new Point(415, 12), Width = 90, Height = 28, BackColor = Color.FromArgb(0, 122, 204), FlatStyle = FlatStyle.Flat };
            btnConnect.Click += BtnConnect_Click;

            pnlConnect.Controls.AddRange(new Control[] { lblName, txtNickname, lblBroker, txtBroker, btnConnect });

            // Messages Log
            rtbMessages = new RichTextBox { Dock = DockStyle.Fill, ReadOnly = true, BackColor = Color.FromArgb(20, 20, 20), ForeColor = Color.LightGray, Font = new Font("Consolas", 10F) };

            // Input Panel
            Panel pnlInput = new Panel { Dock = DockStyle.Bottom, Height = 55, BackColor = Color.FromArgb(45, 45, 45) };
            txtMessage = new TextBox { Location = new Point(10, 15), Width = 480, BackColor = Color.FromArgb(60,60,60), ForeColor = Color.White };
            txtMessage.KeyDown += (s, e) => { if(e.KeyCode == Keys.Enter) { e.SuppressKeyPress = true; btnSend.PerformClick(); } };
            
            btnSend = new Button { Text = "Send", Location = new Point(500, 12), Width = 150, Height = 28, BackColor = Color.ForestGreen, FlatStyle = FlatStyle.Flat, Enabled = false };
            btnSend.Click += BtnSend_Click;
            pnlInput.Controls.AddRange(new Control[] { txtMessage, btnSend });

            this.Controls.AddRange(new Control[] { rtbMessages, pnlConnect, pnlInput });
        }

        private void BtnConnect_Click(object? sender, EventArgs e)
        {
            if (!_isConnected)
            {
                try
                {
                    _producer = new ProducerBuilder<Null, string>(new ProducerConfig { BootstrapServers = txtBroker.Text }).Build();
                    _isConnected = true;
                    btnConnect.Text = "Disconnect";
                    btnSend.Enabled = true;
                    _cts = new CancellationTokenSource();
                    Task.Run(() => StartConsumerLoop(_cts.Token));
                    AppendLog("SYSTEM", "Connected to Kafka Broker.");
                }
                catch (Exception ex)
                {
                    AppendLog("ERROR", $"Connect Error: {ex.Message}");
                }
            }
            else
            {
                _isConnected = false;
                _cts?.Cancel();
                _producer?.Dispose();
                _producer = null;
                btnConnect.Text = "Connect";
                btnSend.Enabled = false;
                AppendLog("SYSTEM", "Disconnected.");
            }
        }

        private async void BtnSend_Click(object? sender, EventArgs e)
        {
            if (!_isConnected || _producer == null || string.IsNullOrWhiteSpace(txtMessage.Text)) return;
            string msg = $"[{DateTime.Now:HH:mm:ss}] {txtNickname.Text}: {txtMessage.Text}";
            txtMessage.Clear();
            try
            {
                await _producer.ProduceAsync(Topic, new Message<Null, string> { Value = msg });
            }
            catch (Exception ex)
            {
                AppendLog("ERROR", $"Produce Exception: {ex.Message}");
            }
        }

        private void StartConsumerLoop(CancellationToken token)
        {
            try
            {
                using var consumer = new ConsumerBuilder<Ignore, string>(new ConsumerConfig { BootstrapServers = txtBroker.Text, GroupId = "winforms-" + Guid.NewGuid() }).Build();
                consumer.Subscribe(Topic);
                while (!token.IsCancellationRequested)
                {
                    var res = consumer.Consume(token);
                    if (res != null)
                    {
                        this.BeginInvoke(new Action(() => {
                            AppendLog("MESSAGE", res.Message.Value);
                        }));
                    }
                }
                consumer.Close();
            }
            catch (Exception) {}
        }

        private void AppendLog(string type, string msg)
        {
            rtbMessages.SelectionStart = rtbMessages.TextLength;
            if (type == "SYSTEM") rtbMessages.SelectionColor = Color.Yellow;
            else if (type == "ERROR") rtbMessages.SelectionColor = Color.LightCoral;
            else rtbMessages.SelectionColor = Color.White;
            rtbMessages.AppendText(msg + "\n");
            rtbMessages.ScrollToCaret();
        }
    }
}
```
